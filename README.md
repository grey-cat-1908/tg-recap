# #TG-RECAP

Узнай, как ты провел этот год в Telegram!

## Использование

### 1. Установка

По-умолчанию предложено два способа установки на выбор:

#### 1) Использование готовых сборок

Вы можете использовать [готовые сборки](https://github.com/grey-cat-1908/tg-recap/releases) программы:

|                                          Windows                                          | macOS |
| ----------------------------------------------------------------------------------------- | ----- |
| [Скачать](https://github.com/grey-cat-1908/tg-recap/releases/download/1.0.0/tg_recap.exe) | [Скачать](https://github.com/grey-cat-1908/tg-recap/releases/download/1.0.0/tg_recap_macos.zip) |

**Запуск на Mac OS:**
При открытии файла компьютер может выдать ошибку из-за того, что приложение не проверено Apple.
<img width="291" height="265" alt="first" src="https://github.com/user-attachments/assets/43d60080-43d5-4dc2-a2ce-12b222ebda34" />
В таком случае нужно:
1) Нажать «Готово»;
2) Открыть в «Системные настройки»;
3) Перейти в раздел «Конфиденциальность и безопасность»;
4) В блоке «Безопасность» нажать кнопку «Все равно открыть».

#### 2) Самостоятельный запуск с помощью Python

### 2. Экспорт истории чата в Telegram

1) Откройте необходимый чат в приложении Telegram для ПК;
2) Нажмите на «три точки» -> «Экспорт истории чата»;
3) Уберите все галочки, поставьте ограничение максимальным;
4) Выберите «Формат» -> «Машиночитаемый JSON»;
5) Укажите путь для сохранения файла;
6) Нажмите на «первого сообщения» и выберите `01.01.2025`;
7) Нажмите «Экспортировать».

### 3. Просмотр карточек

1) Запустите **#TG-RECAP**;
2) Нажмите «Начать» и выберите сохраненный `result.json`;
3) Пользуйтесь :)

*Примечание: Карточки можно сохранить специальной кнопкой, чтобы в дальнейшем поделиться ими в Telgram Stories!*
